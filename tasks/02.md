# Template de Prompt para Criar uma Rota de API

Crie a rota `PUT /workout-plans/{id(uuid)}/days/{id(uuid)}/sessions/:id(uuid)`

## Descrição

Atualiza uma sessão de treino específica.

## Autenticação

- Rota protegida.
- O usuário precisa ser dono do plano de treino recebido.

## Request

```ts
interface Body {
  completedAt: string; // ISO String
}
```

```ts
interface Params {}
```

```ts
interface Query {}
```

## Response

```ts
interface StatusCode201 {
  id: string;
  completedAt: string; // ISO String
  startedAt: string; // ISO String
}
```
