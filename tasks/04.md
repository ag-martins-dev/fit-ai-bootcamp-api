# Template de Prompt para Criar uma Rota de API

Crie a rota `GET /workout-plans/:id`

## Descrição

Essa rota deve retornar o plano de treino recebido e seus dias. Para os dias do plano de treino, NÃO RETORNE OS `workoutExercises`, mas apenas `exercisesCount`.

## Requisitos Técnicos

N/A

## Autenticação

- Rota protegida.
- O usuário precisa ser dono do plano de treino recebido.

## Request

```ts
interface Body {}
```

```ts
interface Params {
  date: string;
}
```

```ts
interface Query {}
```

## Response

```ts
interface StatusCode201 {
  id: string;
  name: string;
  workoutDays: Array<{
    id: string;
    weekDay: string;
    name: string;
    isRest: boolean;
    coverImageUrl?: string;
    estimatedDurationInSeconds: number;
    exercisesCount: number;
  }>;
}
```
